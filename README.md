# Hostel Management System

A web-based hostel management system developed to manage hostel operations efficiently.

## Features
- User registration and login system
- Room booking and management
- Admin dashboard
- User profile management
- Secure authentication system

## Technologies Used
- HTML, CSS, JavaScript
- PHP (Backend)
- MySQL (Database)

## Project Highlights
- Built REST-like backend structure using PHP
- Implemented authentication and session handling
- Designed responsive and user-friendly interface
- Integrated MySQL database for data management
